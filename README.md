##### Disclaimer:

This program is still in progress and is currently in a **NOT WORKING** condition.

## Installation

```sh
$ git clone https://github.com/binhonglee/TicketingSystem.git
$ cd TicketingSystem
$ ./install.sh
```

## Building and running it

```sh
$ ant build
$ ./bin/main
```

## Uninstallation

```sh
$ ./uninstall.sh
```
